#  Password Generator

Aplicação web para gerar senhas seguras com personalização e indicador de força.

---

##  Funcionalidades

* Gerar senha aleatória
* Escolher tamanho da senha
* Incluir maiúsculas, números e símbolos
* Copiar senha com 1 clique

---

##  Tecnologias

* HTML
* CSS
* JavaScript

---

##  Como usar

Abra o arquivo `index.html` no navegador e gere sua senha.

---

##  Objetivo

Projeto para prática de lógica e manipulação do DOM.
